Tim Berners-Lee: Inventor of the World Wide Web

Introduction

Tim Berners-Lee is credited with inventing the World Wide Web, a revolutionary system of interlinked hypertext documents accessed via the internet.

Key Components of the Invention

  • Hypertext: Refers to text displayed on a computer that links to other text which the reader can access immediately.
  • HTTP (Hypertext Transfer Protocol): The foundation of data communication on the World Wide Web. It is the protocol used for transferring hypertext requests between a web browser and a server.
  • HTML (Hypertext Markup Language): The standard markup language for creating web pages and web applications.
